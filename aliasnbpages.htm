AliasNbPages

AliasNbPages([string alias])

バージョン

1.4

説明

全ページの番号のエイリアスを定義します。ドキュメントが閉じられたときに置き換えられます。 訳者注：自信なし。原文： Defines an alias for the total number of pages. It will be substituted as the document is closed.

パラメータ

alias
エイリアス。 デフォルトの値：{nb}

例

class PDF extends FPDF
{
function Footer()
{
    //下端から1.5 cm に移動
    $this->SetY(-15);
    //フォントを設定。 Arial italic 8
    $this->SetFont('Arial','I',8);
    //現在のページ番号と総ページ数を出力
    $this->Cell(0,10,'Page '.$this->PageNo().'/{nb}',0,0,'C');
}
}

$pdf=new PDF();
$pdf->AliasNbPages();
